# Code of conduct

## The short version

Be decent. Assume good faith. Disagree about code, not people.

## What's not OK

- Harassment, insults, or personal attacks.
- Discrimination based on identity of any kind.
- Sustained disruption of discussions or review.
- Publishing someone's private information without their consent.

## Enforcement

If someone is making this project a worse place to participate in, let me know: open a private security advisory (https://github.com/XeldarAlz/FFXIV-Aetherphone/security/advisories/new) or email the address on my GitHub profile. I'll look into it.

Consequences range from a private word, to removing comments, to blocking the account from the repo. Depends on what happened and whether it keeps happening.

## Scope

This applies to issues, pull requests, discussions, and anywhere else tied to this project.
